RVS_MaskButton

@IBDesignable
open class RVS_MaskButton : UIButton

This class can be displayed with either the text filled with a gradient, or the background filled, and the text “cut out” of it. All behavior is the same as any other UIButton.

This allows you to specify a border, which will be included in the gradient fill. If the borderWidth value is anything greater than 0, there will be a border, with corners specified by cornerRadius. The border will be filled with the gradient, as well as the text or image.
